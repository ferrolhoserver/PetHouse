CHANGELOG v1.0:
================

✅ Estrutura completa do aplicativo
✅ 6 abas de prontuário (Peso, Cuidados, Consultas, Cirurgias, Diagnósticos, Tratamentos)
✅ Grid 2x3 de abas com design moderno
✅ Módulo Cuidados com sub-barra de 3 botões (Vacina, Vermífugo, Banho/Tosa)
✅ Sistema de alertas inteligente para vacinas e vermífugos
✅ Exportações: JSON (backup), ICS (calendário), PDF (prontuário)
✅ PWA completo com ícones e manifest
✅ LocalStorage para persistência de dados
✅ Interface responsiva para mobile
✅ Paleta de cores personalizada
✅ Sistema de backup/restauração

- Cadastro de múltiplos pets
- Controle de peso com gráfico
- Registro de vacinas com alertas automáticos
- Registro de vermífugos com lembretes trimestrais
- Registro de banho/tosa
- Registro de consultas veterinárias
- Registro de cirurgias
- Registro de diagnósticos
- Registro de tratamentos
- Cálculo automático de idade
- Alertas de vacinas vencidas
- Exportação de calendário (ICS)
- Exportação de backup completo (JSON)
- Geração de PDF do prontuário
- Instalação como PWA (app nativo)
